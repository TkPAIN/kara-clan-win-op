❓ Frequently Asked Questions

🔹 Are these scripts safe? ▼
Yes! All scripts are open source and only modify Windows settings that can be reverted with the Reset script. No personal data is collected.
🔹 Will this void my warranty? ▼
No. These are standard Windows optimizations that Microsoft itself recommends for gaming/performance.
🔹 Can I undo the changes? ▼
Absolutely! Use the RESET script to revert everything to original Windows settings.
🔹 Do I need to run as Administrator? ▼
Yes, PowerShell must be run as Administrator to modify system settings.
🔹 Does it work on Windows 10/11? ▼
Yes, both Windows 10 and Windows 11 are fully supported.
🔹 Why restart after running? ▼
Some changes require a reboot to take effect properly.
